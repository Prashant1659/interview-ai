# Interview AI

Interview AI is a full-stack AI-powered mock interview platform built with separate `frontend` and `backend` modules to support interview practice, feedback workflows, and scalable web application development. The repository is publicly available on GitHub, currently maintained by a single contributor, and is primarily built with JavaScript and SCSS. 

## Overview

This project is designed as a full-stack web application for AI-assisted interview preparation. The repository structure includes dedicated `backend/` and `frontend/` folders, which indicates a clean separation between client-side and server-side responsibilities and makes the codebase easier to scale, test, and maintain.

## Getting Started

Because the repository is organized into separate frontend and backend modules, setup will likely require installing dependencies in both directories independently. 

1. Clone the repository.
2. Open the project folder.
3. Move into the `backend` folder and install dependencies.
4. Move into the `frontend` folder and install dependencies.
5. Start both applications using the scripts defined in each module.

Example setup flow:

```bash
git clone https://github.com/Prashant1659/interview-ai.git
cd interview-ai

cd backend
npm install
npm run dev
```

Open a second terminal and run:

```bash
cd frontend
npm install
npm run dev
```

> Update the commands above if your actual `package.json` scripts use `npm start`, `vite`, `nodemon`, or any custom script names.
